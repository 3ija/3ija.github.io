HTML

HTML ili Hyper text Markup Language služi za kreiranje web stranica koje se često sastoje od linkova, slika, tekstova, lista i još mnogo toga. Zajedno sa CSS-om i JavaScriptom omogućava kreiranje jedne funkcionalne web stranice.

CSS

CSS ili Cascade Style Sheets koristimo kada želimo da naš HTML dokument ulepšamo i damo mu odredjen izgled od podešavanja boja, margina, ramova, pa sve do manipulisanja slikama, video snimcima i paragrafima.

JS

JavaScript ili skraćeno JS predstavlja glavni izvor funkcionalnosti jedne web stranice. On nam omogućava da kreiramo funkcije koje će izvršavati neku radnju na klik, prilikom učitavanja web stranice, unosa odredjenog teksta itd.
